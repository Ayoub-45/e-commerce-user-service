# e-commerce-user-service

## Overview

The **User Service** is a microservice responsible for user management in the E-Commerce Microservices Platform.  
It handles **registration, login, authentication, and role-based access control (user/admin)**.  
This service is built with **Node.js, Express, and MongoDB**, and it is **dockerized** for easy deployment in a microservices architecture.

## Features

- User Registration with email and password
- Login with JWT token authentication
- Role-based access control (user/admin)
- Password hashing for security
- Dockerized for container deployment
- Ready for integration with API Gateway and other microservices

---

## Technology Stack

- **Node.js** – Runtime
- **Express.js** – Web framework
- **MongoDB** – Database (via Mongoose ORM)
- **JWT** – Authentication
- **Bcrypt** – Password hashing
- **Docker** – Containerization
